# Calculus Pizza Demonstration Project

An interactive mathematics teaching tool that explains the principle of circle division in calculus through pizza slice visualization.

## Project Overview

This project is a web-based interactive mathematics teaching tool that helps learners understand basic concepts in calculus through an engaging pizza slice visualization. The project uses HTML5 Canvas for graphics rendering, modern JavaScript for interaction logic, and features a responsive design.

## How to Use

1. **Basic Operations**
   - Open `index.html` in your browser
   - Use the slice count slider to control the number of pizza slices
   - Use the radius slider to adjust the pizza size
   - Click "Animate" to watch the slicing process
   - Click "Reset" to return to initial state
   - Click "Triangle Approximation" to view detailed mathematical principles

2. **Data Display**
   - Real-time display of approximate area calculation
   - Display of actual circle area
   - Calculation and display of error percentage
   - Detailed mathematical formula explanation
