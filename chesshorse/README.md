# 小马闲逛（Knight's Tour）

## 题目
在 N×N 的棋盘上，国际象棋的“马”（Knight，走“日”字）能否从某个起点出发，走遍棋盘上所有格子，且每个格子只走一次？

本游戏默认 4×4（**无解**），玩家可切换到 5×5、6×6、7×7、8×8（国际象棋棋盘）。

## 玩法
1. 选择棋盘大小（默认 4×4）。
2. 点击任意格子作为起点（♞ 出现）。
3. 按“日”字走到绿色提示的格子。
4. 走过的格子会变色，并显示步号（不可再踩）。
5. 目标：覆盖全部格子（N²/N²）。

## 按钮说明
- `规则`：打开规则说明。
- `撤销`：撤销上一步（回到前一个格子）。
- `提示一步`：根据求解器给出一个“能继续通关”的下一步建议（若当前状态已无解，会提示你撤销/重置）。
- `显示答案`：告诉你
  - 从**当前状态继续**是否还能覆盖全盘；
  - 从**起点重新开始**是否存在完整路径（也会给出“任意起点是否可行”的提示）。
- `动画演示`：自动演示一条可行路径（可调速度；再次点击可停止）。
- `重置`：清空当前局面，重新选择起点。
- `音效`：走子音效开/关（会保存到 localStorage）。

## 新春彩蛋
- 顶部随机祝福语。
- 走子时随机出现祝福成语与小烟花特效。

## 记录
游戏会在本地记录你的尝试次数、成功次数、最佳用时、最高覆盖（localStorage）。

## 打开方式
用浏览器打开：`chesshorse/index.html`。
