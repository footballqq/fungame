使用说明：

将这两个文件保存在同一个文件夹中
需要修改单词时，只需编辑 words.json 文件
功能说明：
页面顶部可以选择"英文模式"或"中文模式"
单词卡片随机打乱顺序显示
点击卡片显示完整信息
可以使用按钮切换上一个/下一个单词
显示当前进度
注意：由于浏览器安全限制，需要通过本地服务器来运行这些文件。您可以：

使用 Visual Studio Code 的 Live Server 插件
或使用 Python 简单的 HTTP 服务器：
bash
复制
python -m http.server
或其他本地服务器方案
这样就可以正常加载 JSON 文件了。

需要修改单词表时，只需编辑 words.json 文件，添加或修改单词信息即可，格式需要保持一致。
