# 马互换（3×3 Knight Puzzle）

## 玩法
- 3×3 棋盘四个角各有一匹马：底部两个角是白马（🐴），顶部两个角是黑马（🐎）。
- 目标：把白马移到顶部两个角，同时把黑马移到底部两个角（互换位置）。
- 规则：马只走“日”字（Knight's move），且**不能落到已被占据**的格子。
- 操作：点按棋子 → 再点按绿色标记的目标格；支持撤销、重置、提示一步。

## 新春彩蛋
- 走子时有概率触发“马相关祝福成语”提示。
- 走子时有概率出现烟花特效。
- 顶部有“马年吉祥”祝福条（纯装饰）。
- 支持记录玩家成绩（localStorage）：通关次数、个人最佳、最近通关、最优达成。
- 支持走子音效，可关闭（设置会被保存）。

## 最少步数
最优解步数：**16 步**（游戏内会显示并在通关后提示你是否达到最优）。

## 打开方式
直接用浏览器打开：`horse/index.html`。
