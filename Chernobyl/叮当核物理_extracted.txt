1986年4月25日晚上至26日凌晨。苏联乌克兰普里皮亚季市的切尔诺贝利核电站爆炸了，4号反应堆。
中子撞击铀235的原子核后，会产生钡克和氪和三个中子。石墨是用来减速中子的，这样更容易被燃料棒捕获。氙135会吃中子，吃中子以后会变成氙136。如果中子不够氙135吃的话，反应堆的功率就会持续下降。一直降到了正常功率的5%，这个过程叫反应堆毒化。
加特洛夫是刚刚上任的代理总工程师。他不想停机，他提出了一个方案，让功率升上去。这个方案是，关闭自动控制，拔出控制棒。
但是水可以吸收中子，而且燃料棒周围是有水的。但是水旁边有气泡，气泡的吸收中子的能力没那么强，所以中子就会进入核反应堆的燃料棒里。产生剧烈的核反应，导致功率升高到原来的10倍，自动控制系统如果开启就会把控制棒全部放下，但是自动控制系统被他们断开了。
1:23:04的时候，开始了断电测试。
在1:23:40的时候，工作人员终于发现了不对劲，就赶紧按下了紧急停堆按钮。控制棒落下的速度是每秒0.4米但一共需要落下的高度是7米（大约20秒左右落下）。控制棒本身的材质是硼，但是底下放了一小块石墨，所以在一开始落下的时候，反应堆的活动是加剧的。因为温度太高了，在1/3处的时候。燃料棒发生了形变。所以控制棒卡住了。
于是在1:23:47的时候爆炸了。
4号反应堆虽然停堆了，爆炸了。但是，1号反应堆、2号反应堆和3号反应堆一直在工作，工作到了2000年才正式停堆。
一开始想做一个简单的石棺把它包起来，但它只有30年的寿命。1997年建了一个很大的拱形的棺材，把它永久地包起来，2016年才完工。
4号反应堆的爆炸把重达2000吨的盖子给炸开了。大量高能辐射物质喷上几千米的高空。8吨多的强辐射物进入空气。24万人进入辐射区。

一、核聚变是什么
核聚变：把两个轻原子核（如氢的同位素氘、氚）融合成一个较重的原子核，并释放巨大能量的核反应，也叫热核反应。
核裂变：重核（如铀）拆成两个中等核，释放能量（核电站、原子弹）。
核聚变：轻核（氘、氚）合成重核（氦），释放能量（太阳、氢弹）。
二、核心原理：两种力+质量亏损
1. 两种力的博弈
库仑斥力：原子核都带正电，距离远时互相排斥（长程力）。
强核力（核力）：距离极近（≈米）时，极强的吸引力，远大于库仑斥力（短程力）。 要聚变，必须让轻核靠到强核力范围，克服斥力。
2. 能量来源：质量亏损与
氘+氚→氦+中子，反应后总质量略小于反应前，消失的质量叫质量亏损。
爱因斯坦质能方程：（为光速≈30万公里/秒）。
极小质量→巨大能量：1克氘氚聚变≈燃烧25吨煤。
三、实现聚变的极端条件（缺一不可）
1. 超高温（克服斥力）
太阳核心：1500万℃，靠引力高压维持。
人工聚变（如ITER、东方超环）：1亿–1.5亿℃（太阳核心的10倍），用高温给粒子足够动能撞在一起。
2. 高密度（增加碰撞概率）
把氘氚等离子体（高温带电气体）压缩到足够密，让原子核频繁相撞。
3. 长时间约束（维持反应）
高温等离子体不能碰容器壁（一碰就冷却），用磁场约束（如托卡马克，“甜甜圈”形磁场），让等离子体悬空悬浮，维持足够反应时间。
三者合称劳逊判据：温度×密度×约束时间 达到阈值，才能净输出能量（输出＞输入）。
四、最主流反应：氘–氚聚变（D–T）
燃料：氘（重氢） + 氚（超重氢）。
反应式：
氘：海水里有，1升海水≈300升汽油能量，几乎取之不尽。
氚：自然界极少，可用锂在反应堆中人工生产。
产物：氦（惰性、无放射性） + 高能中子，无长寿命核废料，清洁安全。
五、太阳：天然聚变反应堆
核心：1500万℃、3000亿大气压，引力压缩氢核，持续质子–质子链聚变。
过程：4个氢核→1个氦核，质量亏损→光和热，每秒释放能量≈人类全年能耗的100万倍。
六、人工可控核聚变：难点与进展
1. 难点
1亿℃高温，材料无法承受，只能用磁场约束。
维持稳定：等离子体极不稳定，易“逃逸”或冷却。
能量净输出：输入加热/约束能量＜输出聚变能量（“点火”）。
2. 进展
中国东方超环（EAST）：实现1.2亿℃、403秒长脉冲高温等离子体运行，世界领先。
ITER（国际热核聚变实验堆）：中法等7方共建，2025–2035年实验目标：净能量输出（500MW输出，持续400秒）。
七、聚变能：终极能源的优势
燃料几乎无限：氘来自海水，锂可产氚，够用数十亿年。
清洁无排放：无CO₂、无酸雨、无长寿命核废料。
安全：无链式反应，一旦约束失效，等离子体瞬间冷却，不会爆炸或熔堆。
一句话总结：核聚变就是把轻核“烧”成重核，用一点点质量换无穷清洁能源，人类正在努力把“太阳”装到地球上。

好，我直接把“怎么用锂造出氚（那个超重氢）”讲得又科学又通俗，一步到位。

1. 你要造的是：氚（³H，T）
氚 = 氢的一种同位素，原子核里：1个质子 + 2个中子
自然界几乎没有氚，只能靠用中子打锂造出来。

2. 核心反应（最主要、最实用）
用慢中子 n 打 锂6（⁶Li）：
⁶Li + n → ⁴He（氦） + ³H（氚） + 4.8 MeV 能量
简单说：
一颗锂6 + 一颗中子 → 一颗氦 + 一颗氚 + 放热
还有一个次要反应（用锂7）：
⁷Li + n → ⁴He + ³H + n −2.5 MeV
也要高能中子，而且要吸热，一般当补充，不做主产。

3. 实际怎么做（工业/未来聚变堆）
① 准备“锂靶”
把锂做成：
金属锂
或锂铝合金、氟化锂、碳酸锂
或未来聚变堆用的锂陶瓷小球（如 Li₂TiO₃、Li₄SiO₄）
② 用中子去“打”锂
中子来源两种：
现在：裂变反应堆（铀裂变放出大量中子）
未来聚变电厂：氘氚聚变自己放出的 14.1 MeV 高能中子 → 直接打在包层的锂上，现场造氚
③ 把生成的氚“抽出来”
氚是气体（和氢气很像），流程：
锂材料被中子辐照一段时间
加热（几百度），氚会从锂里跑出来
用钯管/铀吸附/热扩散等方法提纯到 99% 以上

4. 为什么这能解决“氚不够用”
氘氚聚变：D + T → He + n(14.1MeV)
这个中子立刻去打锂：n + ⁶Li → He + T
理想情况：烧掉1个氚，又造出≥1个氚 → 氚可以循环自给自足

5. 一句话总结
用中子轰击锂6，就能把锂变成氦＋氚；氚再拿来和氘聚变，聚变放出的中子又去造新的氚，形成闭环。
